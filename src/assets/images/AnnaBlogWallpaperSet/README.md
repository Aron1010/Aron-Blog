# Anna blog wallpaper set

All wallpaper assets in this directory are prepared for the existing responsive blog layout.

- `selected-desktop-*`: selected from the supplied images, cropped and resized to `2688x1536`.
- `selected-mobile-*`: selected from the supplied images, cropped and resized to `3328x4864`.
- `generated-desktop-*` / `generated-mobile-*`: newly generated variants, then softened and encoded to AVIF to better match an anime episode frame.

The files use AVIF to match the project's existing wallpaper assets and reduce transfer size.
